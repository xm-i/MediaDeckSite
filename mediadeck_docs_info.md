# MediaDeck ドキュメントWebサイト — 情報まとめ

## 1. アプリケーション概要

| 項目 | 内容 |
|---|---|
| 名前 | MediaDeck |
| 種別 | Windows向け高機能メディア管理アプリケーション |
| 対応OS | Windows 10 (1809) 以降 |
| ランタイム | .NET 10 |
| UIフレームワーク | WinUI 3 (Windows App SDK) |
| ライセンス | MIT |
| 言語対応 | 日本語 / 英語（アプリ内で切り替え可能、システム既定にも追従） |
| リポジトリ | `https://github.com/xm-i/MediaDeck` |

**一言説明**: 画像、動画、PDF、アーカイブファイルなどを一元管理し、高度な検索・タグ付け・メタデータ管理機能を提供するデスクトップアプリ。

---

## 2. 対応メディアタイプ

| カテゴリ | 対応フォーマット |
|---|---|
| **画像** | JPEG, PNG, GIF, BMP, TIFF, HEIF, PSD, RAW (RAF), ICO, PCX, Netpbm 等 |
| **動画** | MP4, MKV, AVI, MOV 等 (FFMpeg がサポートする形式) |
| **PDF** | Windows.Data.Pdf による高速表示・サムネイル生成 |
| **アーカイブ** | ZIP, 7z, RAR 等 |

## 3. UI構成（メインウィンドウのレイアウト）

メインウィンドウは以下のゾーンで構成されている:

```
┌──────────────────────────────────────────────────────┐
│ [アイコン] [ナビゲーションメニュー]    [通知ボタン]     │ ← タイトルバー
├──────────────────────────────────────────────────────┤
│ [Tab 1] [Tab 2] [+]                  [InfoBar]       │ ← タブバー
├────────────┬─────────────────────────┬───────────────┤
│ リポジトリ  │                         │               │
│ ペイン      │                         │ 詳細ペイン     │
│ (フォルダ/  │     ビューアペイン       │ (プロパティ/   │
│  アルバム)  │   (グリッド/リスト/      │  タグ)        │
├────────────┤    詳細/マップ)          │               │
│ ソート      │                         │               │
│ ペイン      │                         │               │
├────────────┤                         │               │
│ フィルタ    │                         │               │
│ ペイン      │                         │               │
├────────────┴─────────────────────────┴───────────────┤
│ ステータスバー    [バックグラウンドタスク表示]           │
└──────────────────────────────────────────────────────┘
```

- 各ペイン間は **GridSplitter** でリサイズ可能
- ペインの幅・高さはタブごとに保存され、新規タブにはデフォルト値が適用される
- ウィンドウ背景は **DesktopAcrylicBackdrop**（半透明アクリル効果）

## 4. 主要機能の詳細

### 4.1 タブ機能
- **複数タブ対応**: ブラウザのようにタブを追加・削除して並行管理
- **タブの名前変更**: ダブルクリック or コンテキストメニュー「名前の変更」
- **タブの移動**: コンテキストメニューから「新しいウィンドウで開く」「別のウィンドウに移動」
- **マルチウィンドウ**: 複数ウィンドウを開いてタブを分離管理
- **タブ状態の永続化**: ソート条件、フィルタ条件、表示設定などがタブごとに保存される
- 新しいタブは「デフォルトタブ状態」（最後に操作したタブの設定）を引き継ぐ

### 4.2 リポジトリペイン（左上）
2つのモードを切り替え可能:
- **フォルダリポジトリ**: ファイルシステムのディレクトリ構造をツリー表示。フォルダを選択して表示範囲を絞り込む。サブフォルダ含む/含まないの選択が可能。
- **アルバムリポジトリ**: ユーザーが自由に作成できる仮想的なグループ。ファイルをドラッグ＆ドロップでアルバムに追加。階層構造（`/`区切り）対応。

### 4.3 ビューアペイン（中央） — 4つのビューモード
| ビュー | 説明 |
|---|---|
| **グリッド表示 (Wrap)** | サムネイルをタイル状に並べる。サイズスライダーで拡大縮小可能。オーバーレイ（ファイル名等）やInfo表示の切り替えあり。 |
| **リスト表示 (List)** | テーブル形式。列: ファイル名、解像度、ファイルサイズ、作成日時、評価。列幅・表示/非表示はカスタマイズ可能。 |
| **詳細表示 (Detail)** | 選択ファイルの大きなプレビューを表示 |
| **マップ表示 (Map)** | 位置情報(GPS)を持つファイルを地図上にピン表示 |

- **検索バー**: ビューア上部に配置。タグ、フォルダ、プロパティ条件などをトークン形式で入力
- **検索条件サジェスト**: 入力中にタグ名、フォルダ名、`prop.`プロパティがサジェストされる

### 4.4 検索・フィルタリング機能

#### 検索バー（トークン型検索）
検索バーにテキストを入力すると、以下の種類の検索条件がサジェストされる:

| 種類 | 入力例 | 説明 |
|---|---|---|
| **タグ検索** | タグ名を入力 | タグ名、読み仮名(Ruby)、ローマ字、エイリアスで検索 |
| **フォルダ検索** | フォルダパスを入力 | 特定フォルダ内のファイルに絞り込み |
| **プロパティ検索** | `prop.` で始まる入力 | 専用ダイアログでプロパティ比較条件を設定 |
| **アルバム検索** | アルバム名を入力 | 特定アルバム内のファイルに絞り込み |
| **アドレス検索** | 住所・地名を入力 | 位置情報に基づく検索 |
| **ワード検索** | 任意テキスト | ファイルパスに対するテキスト検索 |

#### プロパティ検索 (`prop.`) の対象プロパティ一覧

| プロパティ名 | 型 | 利用可能な演算子 |
|---|---|---|
| `Rate` | 数値 | `>`, `>=`, `=`, `<=`, `<` |
| `Width` | 数値 | `>`, `>=`, `=`, `<=`, `<` |
| `Height` | 数値 | `>`, `>=`, `=`, `<=`, `<` |
| `UsageCount` | 数値 | `>`, `>=`, `=`, `<=`, `<` |
| `FileSize` | 数値 | `>`, `>=`, `=`, `<=`, `<` |
| `CreationTime` | 日時 | `>`, `>=`, `=`, `<=`, `<` |
| `ModifiedTime` | 日時 | `>`, `>=`, `=`, `<=`, `<` |
| `LastAccessTime` | 日時 | `>`, `>=`, `=`, `<=`, `<` |
| `RegisteredTime` | 日時 | `>`, `>=`, `=`, `<=`, `<` |
| `MediaType` | Enum | `=` |
| `IsExists` | bool | `=` |
| `IsInvalid` | bool | `=` |
| `IsAutoGeneratedThumbnail` | bool | `=` |
| `IsUnderFolderGroup` | bool | `=` |
| `Description` | 文字列 | `=`, `contains` |
| `DirectoryPath` | 文字列 | `=`, `contains` |
| `FilePath` | 文字列 | `=`, `contains` |

表示形式例: `prop.Width >= 1920`, `prop.Rate > 3`

#### フィルタ条件
フィルタはプリセットとして保存・管理可能。以下のフィルタアイテムを組み合わせ:

| フィルタ種類 | 説明 |
|---|---|
| **タグフィルタ** | 特定タグの有無で絞り込み |
| **ファイルパスフィルタ** | パス文字列で絞り込み |
| **解像度フィルタ** | 幅・高さの範囲で絞り込み |
| **評価フィルタ** | レート値で絞り込み |
| **メディアタイプフィルタ** | 画像/動画/PDF/アーカイブなどで絞り込み |
| **位置情報フィルタ** | GPS情報の有無や範囲で絞り込み |
| **存在フィルタ** | ファイルが実在するか |
| **フォルダグループフィルタ** | フォルダグループ配下かどうか |

- フィルタ条件はフィルタマネージャーウィンドウで管理・編集可能

### 4.5 ソート機能

利用可能なソートキー:

| ソートキー | 説明 |
|---|---|
| FilePath | ファイルパス順 |
| CreationTime | 作成日時順 |
| ModifiedTime | 更新日時順 |
| LastAccessTime | 最終アクセス日時順 |
| RegisteredTime | 登録日時順 |
| FileSize | ファイルサイズ順 |
| Rate | 評価順 |
| Location | 位置情報(緯度)順 |
| Resolution | 解像度(幅×高さ)順 |
| UsageCount | 使用回数順 |

- 昇順/降順の切り替え可能
- 複数ソート条件の組み合わせ（ソートマネージャーで管理）

### 4.6 詳細ペイン（右側）
2つのタブで構成:
- **プロパティタブ**: 選択ファイルのメタデータ表示（ファイル名、解像度、ファイルサイズ、日時等）。複数選択時は集計表示。評価(Rate)の変更、説明(Description)の編集、使用回数の表示。メタデータ情報の閲覧。
- **タグタブ**: 選択ファイルに付与されたタグの表示・追加・削除。タグ入力時のサジェスト（名前、読み仮名、ローマ字、エイリアス対応）。タグクリックで同じタグのファイルを検索。タグマネージャーへの遷移。

### 4.7 タグ管理（タグマネージャー）
- **タグの作成・編集・削除**
- **タグカテゴリ**: タグをカテゴリに分類
- **タグ属性**: タグ名、読み仮名(Ruby)、ローマ字(自動変換)、説明、使用回数
- **タグエイリアス**: 別名の追加（エイリアスでも検索可能）
- 専用ウィンドウ (TagManagerWindow) で管理

### 4.8 フォルダ管理（フォルダマネージャー）
- スキャン対象フォルダの登録・編集・削除
- 専用ウィンドウ (FolderManagerWindow)

### 4.9 重複ファイル検出ツール
- ファイルのハッシュ値に基づく重複検出
- **PreHash**: 高速な部分ハッシュ（概算の重複チェック）
- **FullHash**: 完全ハッシュ（正確な重複チェック）
- 重複グループ数、重複ファイル総数の表示
- 専用ウィンドウ (DuplicateDetectorWindow)

### 4.10 サムネイル管理
- **サムネイル一括再生成**: 専用ウィンドウ (BulkThumbnailRegenerationWindow)
- **サムネイルピッカー**: 動画等のサムネイルを選択する機能 (ThumbnailPickerWindow)

### 4.11 ファイル変更同期
- 外部でのファイル移動・削除・名前変更を検出
- タイトルバーに通知ボタン（パルスアニメーション付き）が表示される
- 専用ウィンドウで変更内容を確認し、「反映」or「無視」を個別/一括で選択
- フィルター（すべて/移動/削除等）で表示を切り替え可能

### 4.12 バックグラウンドタスク
ステータスバーにバックグラウンドタスクの進捗を表示:
- ファイルスキャン・登録
- メタデータ抽出
- サムネイル生成
- ハッシュ計算
- ファイル状態更新

## 6. 設定画面 (ConfigWindow)

設定は以下のページで構成:

| 設定ページ | 内容 |
|---|---|
| **スキャン設定** | 対象リポジトリのパス、スキャン対象の拡張子管理 |
| **検索設定** | 初回読み込み件数、増分読み込み件数、最大読み込み件数 |
| **実行プログラム設定** | メディアタイプごとに外部プログラム（VLC等）での起動設定 |
| **サムネイル設定** | サムネイルのサイズ・品質設定 |
| **言語設定** | 日本語/英語の切り替え（システム既定に追従も可能） |
| **環境設定** | その他の環境設定 |

## 7. 技術的な特徴（ユーザー向けに言い換え）

| 技術 | ユーザー向け説明 |
|---|---|
| IAsyncEnumerable ストリーミング | 大量のファイルでもUIがフリーズせず、逐次表示される |
| SQLite + EF Core | 高速なデータベース検索。数万件でも瞬時にフィルタリング |
| Magick.NET / MetadataExtractor | 画像メタデータ（EXIF等）の自動抽出 |
| FFMpegCore | 動画メタデータの自動抽出・サムネイル生成 |
| DesktopAcrylicBackdrop | モダンな半透明アクリル背景 |
| マルチウィンドウ・マルチタブ | 複数の作業を並行して効率的に管理 |
